Derivative Rules Reference

Rule Name Formula Example Description
Power Rule
d/dx[x^n] = n路x^(n-1)
d/dx[x^5] = 5x^4
d/dx[x^3] = 3x^2
d/dx[x^(-2)] = -2x^(-3)
Used to find the derivative of functions where x is raised to a constant power. Multiply by the exponent and reduce the exponent by 1.
Product Rule
d/dx[f(x)路g(x)] = f'(x)路g(x) + f(x)路g'(x)
d/dx[x^2路sin(x)] = 2x路sin(x) + x^2路cos(x)
Used when differentiating the product of two functions. Take the derivative of the first times the second, plus the first times the derivative of the second.
Linear Operator Rule
d/dx[af(x) + bg(x)] = a路f'(x) + b路g'(x)
d/dx[3x^2 + 5x] = 3(2x) + 5(1) = 6x + 5
The derivative of a linear combination equals the linear combination of the derivatives. Constants can be factored out of derivatives.
Exponent Rule
d/dx[e^(f(x))] = e^(f(x))路f'(x)

d/dx[a^(f(x))] = a^(f(x))路ln(a)路f'(x)
d/dx[e^(2x)] = e^(2x)路2 = 2e^(2x)
d/dx[3^x] = 3^x路ln(3)
Used for exponential functions. For e^x, multiply by the derivative of the exponent. For other bases, include ln(base) as a factor.
Derivative of Logarithm Rule
d/dx[ln(f(x))] = f'(x)/f(x)

d/dx[log_a(f(x))] = f'(x)/(f(x)路ln(a))
d/dx[ln(x^2)] = 2x/x^2 = 2/x
d/dx[log_10(x)] = 1/(x路ln(10))
For natural logarithm, divide the derivative of the argument by the argument itself. For other bases, include 1/ln(base) as a factor.
